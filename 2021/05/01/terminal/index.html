Pushd and Popd Commands in Linux

Script for directory favorite.

pushd: pushd [-n] [+N | -N | dir]
Add directories to stack.

pushd

# Change into and add to the stack.
pushd -n # Add to the stack, not change into it.

To better illustrate the options, let’s print the current directory stack:

dirs -l -v

pushd +N | -N # allows you to navigate to Nth/reverse Nth directory of the stack

popd: popd [-n] [+N | -N]
Remove directories from stack.
